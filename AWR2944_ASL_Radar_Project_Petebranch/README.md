# Radar Data Visualization System

A Python-based system for visualizing radar data from TI mmWave sensors (AWR2944, etc.).

## Features
- Real-time Range-Doppler map visualization
- 3D point cloud display
- Ethernet data capture support
- Serial configuration for TI radar sensors
- Modular signal processing pipeline
